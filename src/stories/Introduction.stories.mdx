# Introduction

`react-responsive-slider` is a carousel-style slider that supports CSS-based
responsive designs, multiple items per slide and has built-in lightbox support
(using [react-modal][modal]) with pinch-zoom. It uses [useGesture][gesture]
under the hood for mouse/touch gestures and [react-spring][spring] for
animations.

[modal]: https://reactcommunity.org/react-modal/
[gesture]: https://use-gesture.netlify.app/
[spring]: https://react-spring.io/

## Features
* use standard CSS without JS magic for dimensioning
* responsive functionality trough standard CSS (media queries)
* built in lightbox support that utilizes the same slider component internally
* pinch zoom functionality
* autoplay

The slider doesn't control the responsive behavior, rather it tracks size
changes and follows the sizing set by the CSS. This enables the usage of
standard media queries inside CSS to, for example, change the number of items on
a slide. By default the slider displays one item per slide and you need to
implement the responsive functionality based on your needs. All of the styling
is applied trough CSS classes, which enables easy integration with, i.e.
[tailwind css][tw].

[tw]: https://tailwindcss.com/
